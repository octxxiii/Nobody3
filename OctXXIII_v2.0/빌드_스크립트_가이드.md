# OctXXIII v2.0 빌드 스크립트

## 🛠️ 자동 빌드 스크립트

이 스크립트는 OctXXIII 애플리케이션을 자동으로 빌드하고 배포 준비를 도와줍니다.

## 📋 빌드 과정

### 1. 환경 확인
- Python 3.8+ 설치 확인
- 필요한 패키지 설치
- FFmpeg 다운로드 및 설정

### 2. 실행파일 빌드
- cx_Freeze를 사용한 실행파일 생성
- FFmpeg 바이너리 포함
- 최적화된 빌드 설정

### 3. 배포 준비
- 사용자 문서 생성
- 배포용 폴더 구성
- 실행 테스트

## 🚀 사용 방법

### Windows에서 빌드
```bash
# 1. Python 설치 확인
python --version

# 2. 의존성 설치
pip install -r requirements.txt
pip install cx_Freeze

# 3. 빌드 실행
python build_windows.py
```

### 수동 빌드 (대안)
```bash
# FFmpeg 수동 다운로드 후
python setup.py build
```

## ⚙️ 빌드 설정

### setup.py 주요 설정
```python
# 실행파일 설정
executables = [
    Executable(
        "Nobody3.py",
        base="Win32GUI",
        target_name="OctXXIII.exe",
        icon="icon.ico"
    )
]

# 포함할 패키지
packages = ["PyQt5", "yt_dlp", "requests", "urllib3", "certifi"]

# 포함할 파일
include_files = [
    "resources_rc.py",
    ("ffmpeg/windows/ffmpeg.exe", "ffmpeg.exe"),
    ("ffmpeg/windows/ffprobe.exe", "ffprobe.exe"),
]
```

## 🔧 문제 해결

### 빌드 오류
1. **Python 버전**: 3.8 이상 필요
2. **의존성**: `pip install --upgrade -r requirements.txt`
3. **권한**: 관리자 권한으로 실행
4. **공간**: 최소 2GB 여유 공간 필요

### 실행 오류
1. **FFmpeg**: 실행파일과 같은 폴더에 있어야 함
2. **바이러스 백신**: 실행파일 차단 확인
3. **의존성**: Visual C++ Redistributable 설치
