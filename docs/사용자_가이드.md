# OctXXIII v2.0 사용자 가이드

## 📖 개요
OctXXIII는 YouTube, YouTube Music, SoundCloud 등의 플랫폼에서 비디오와 오디오를 다운로드하고 재생할 수 있는 통합 애플리케이션입니다.

## 🚀 주요 기능

### ✨ 2025년 업데이트 기능
- **미니 플레이어 모드**: 최소화 시 자동으로 미니 플레이어로 전환
- **최상위 고정 토글**: 미니 플레이어를 항상 위에 표시
- **최대화 버튼 활성화**: 창 크기 조절 기능 개선
- **FFmpeg 포함 빌드**: 별도 설치 없이 바로 사용 가능
- **크로스 플랫폼 지원**: Windows, macOS, Linux 지원

### 🎵 핵심 기능
- **통합 브라우저**: YouTube, YouTube Music, SoundCloud 내장 브라우저
- **다양한 포맷 지원**: MP3, MP4, WebM, M4A 등 다양한 포맷
- **플레이리스트 지원**: 전체 플레이리스트 다운로드
- **실시간 미디어 컨트롤**: 재생/일시정지, 이전/다음 트랙
- **포맷 설정**: 사용자 맞춤형 포맷 필터링
- **다크 테마**: 눈에 편한 다크 인터페이스

## 📥 설치 방법

### 방법 1: 실행파일 다운로드 (권장)
1. 릴리즈 페이지에서 `OctXXIII.exe` 다운로드
2. 다운로드한 파일을 원하는 위치에 저장
3. 더블클릭하여 실행

### 방법 2: 소스코드에서 빌드
```bash
# 1. Python 설치 (3.8 이상)
# 2. 의존성 설치
pip install -r requirements.txt

# 3. 실행
python Nobody3.py
```

## 🎮 사용 방법

### 기본 사용법
1. **애플리케이션 실행**: `OctXXIII.exe` 더블클릭
2. **브라우저에서 영상 선택**: 내장 브라우저에서 원하는 영상 찾기
3. **URL 복사**: 📋 버튼 클릭하여 현재 URL 자동 복사
4. **포맷 선택**: 테이블에서 원하는 포맷 선택
5. **다운로드**: 📥 버튼 클릭하여 다운로드 시작

### 고급 기능

#### 미니 플레이어 모드
- 창을 최소화하면 자동으로 미니 플레이어로 전환
- 미니 플레이어에서 재생 컨트롤 가능
- 📌 버튼으로 최상위 고정 토글
- 🔼 버튼으로 원래 크기로 복원

#### 포맷 설정
- ⚙️ 버튼 클릭하여 포맷 설정 창 열기
- 기본 포맷 선택 (mp3, mp4, webm, m4a, best)
- 표시할 포맷 타입 선택 (비디오, 오디오, 오디오 전용)
- 최대 품질 제한 설정 (480p, 720p, 1080p, 무제한)

#### 브라우저 컨트롤
- 👈👉 이전/다음 페이지 이동
- 🔄 새로고침
- 🏠 YouTube 홈
- 🎵 YouTube Music
- 🎧 SoundCloud
- 🦕 브라우저 숨기기/보이기
- 💥 다운로드 패널 숨기기/보이기

## ⚙️ 설정 및 커스터마이징

### 포맷 설정
- **기본 포맷**: 다운로드 시 기본 선택되는 포맷
- **표시 필터**: 원하지 않는 포맷 타입 숨기기
- **품질 제한**: 최대 해상도 제한 설정

### 캐시 관리
- 💬 버튼으로 정보 창 열기
- 캐시 크기 확인 및 정리
- 설정 자동 저장
