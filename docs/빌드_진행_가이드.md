# OctXXIII 빌드 진행 가이드

## 현재 상황

- ✅ 아이콘 및 앱 이름 설정 완료
- ✅ `icon.icns` 파일 생성 완료
- ⚠️ Python이 PATH에 없음 (빌드를 위해 필요)

## 빌드 방법

### 방법 1: Python PATH 설정 후 빌드 (권장)

1. **Python 설치 확인**
   ```powershell
   # Python이 설치되어 있는지 확인
   py --version
   # 또는
   python3 --version
   ```

2. **Python 경로 찾기**
   - 일반적으로 설치 위치:
     - `C:\Python3x\` 
     - `C:\Users\[사용자명]\AppData\Local\Programs\Python\`
     - `C:\Program Files\Python3x\`

3. **PATH에 Python 추가**
   - Windows 설정 → 시스템 → 고급 시스템 설정 → 환경 변수
   - 시스템 변수 `Path`에 Python 설치 경로 추가

4. **빌드 실행**
   ```cmd
   python build_windows.py
   ```
   또는
   ```cmd
   python setup.py build
   ```

### 방법 2: Python 전체 경로 사용

Python이 설치되어 있지만 PATH에 없는 경우:

```powershell
# Python 전체 경로로 빌드
C:\Python312\python.exe setup.py build

# 또는 py 런처 사용
py setup.py build
```

### 방법 3: 가상 환경 사용

```powershell
# 가상 환경 생성 (Python이 설치되어 있는 경우)
python -m venv venv

# 가상 환경 활성화
.\venv\Scripts\Activate.ps1

# 의존성 설치
pip install -r requirements.txt
pip install cx_Freeze

# 빌드
python setup.py build
```

## Windows용 아이콘 파일 생성

빌드 전에 Windows용 `icon.ico` 파일을 생성하는 것이 좋습니다:

```powershell
# Pillow 설치 필요
pip install Pillow

# 아이콘 생성
python create_windows_icon.py
```

## 빌드 결과

빌드가 완료되면 다음 폴더에 실행 파일이 생성됩니다:

- `build_new/OctXXIII.exe` (기존 빌드)
- `build/exe.win-amd64-3.x/OctXXIII.exe` (새 빌드)

## 이미 빌드된 파일 사용

현재 `build_new` 폴더에 이미 빌드된 파일이 있습니다:
- `build_new\OctXXIII.exe`

새 빌드가 필요하지 않다면 이 파일을 사용할 수 있습니다.

## 문제 해결

### Python이 없는 경우

1. Python 다운로드 및 설치
   - https://www.python.org/downloads/
   - 설치 시 "Add Python to PATH" 체크!

2. 재설치
   - PATH 추가를 빠뜨린 경우, Python 재설치 시 반드시 체크

### cx_Freeze 오류

```powershell
pip install --upgrade cx_Freeze
```

### 빌드 오류

```powershell
# 의존성 재설치
pip install -r requirements.txt --force-reinstall
pip install cx_Freeze --force-reinstall
```
